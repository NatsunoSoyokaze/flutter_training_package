# 概要
本件は Flutter を用いたカメラ機能をまとめました。
以下カメラ機能を実装しています。
1. カメラ撮影
2. 撮影画像プレビュー表示
3. 撮影画像保存
4. 動画撮影
5. 撮影動画プレイ表示
6. 撮影動画保存
7. イン・アウト切り替え

※（今後予定、フラッシュ、撮影画像のフィルタ、音楽付与など）

※ 画面自体は絶望的です

## 前提条件
Android Studio で作成のため、VSCode、その他 IDE での検証はありません。
（特に問題ないと思いますが）
なお、iOS シミュレータはカメラ機能がないため動作確認はできません。

### 使用パッケージ
- [camera](https://pub.dev/packages/camera)
- [permission_handler](https://pub.dev/packages/permission_handler)

### 使い方
1. permission_handler.dart を import してください。
2. カメラを起動するタイミングで以下画面を呼び出してください。

``` Flutter
CameraScreen();
```

### 留意点
サービス側でカメラの権限をチェックしています。
権限がない場合はキャンセル or 設定アプリに遷移する実装を行なっています。

そのため、アプリ側で権限のチェックを行う必要はありません。
