# 概要
学習用プロジェクトです。

最低限の機能のみ使いやすいよう実装をしてます。
（UI は実装していないため、オリジナルしやすいよう設計してます）

## 機能
 - カメラ機能
   - 写真撮影
     - 保存機能

※ フィルタリング等画像加工を予定
   - 動画撮影
     - 保存機能

 　　※ 音楽追加など予定
 - 写真・ギャラリー
   - 端末内の写真取得 
   - ギャラリー表示の基本実装

※ シンプルな実装を重視しています

 - 位置情報
   - 現在位置の取得 
   - 緯度 / 経度の取得
 - スタッフロール（個人趣味） 
   - 学習目的と趣味を兼ねて、エンジニアとして制作したことを表現するために実装しています。
